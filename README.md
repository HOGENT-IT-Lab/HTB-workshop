# HTB workshop

Dit is de repository voor de HackTheBox workshop van het IT-lab. 

De slides van deze workshop kan je [hier](https://hogent-it-lab.github.io/HTB-workshop/slides) vinden en achteraf raadplegen.

## Opzet workshop

Tijdens deze workshop maak je kennis met het HackTheBox-platform. We voorzien een korte introductie van de mogelijkheden en use cases van het platform. Daarnaast geven we je ook een quickstart guide en reiken we je enkele principes en tools aan die je kunnen helpen bij het experimenteren en leren werken met het platform.
